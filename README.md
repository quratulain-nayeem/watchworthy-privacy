# watchworthy-privacy
Privacy policy for the WatchWorthy Chrome extension.
WatchWorthy Privacy Policy

WatchWorthy helps users analyze YouTube videos for learning quality, transcript-based Q&A, and related recommendations.

Data processed:
WatchWorthy may process the current YouTube video URL or video ID, available English transcript/caption text, video metadata, user-entered questions, and user-entered recommendation queries.

How data is used:
This data is sent to the WatchWorthy backend only to generate video scoring, answers, and recommendations. The data is not sold, used for advertising, or used to determine creditworthiness.

Data sharing:
WatchWorthy uses third-party services needed to provide the extension’s functionality, including Google Cloud infrastructure, YouTube APIs, and Gemini API. Data is shared only as necessary to provide the requested analysis.

Data retention:
WatchWorthy does not intentionally store personal user profiles. Temporary processing and service logs may be retained by infrastructure providers for security, debugging, and operational purposes.
